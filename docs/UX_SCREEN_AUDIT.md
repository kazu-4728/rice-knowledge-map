## 凡例

| 項目 | 説明 |
|---|---|
| 役割 | この画面が担う唯一の目的 |
| 主利用者 | 想定される主要利用者 |
| 最初の主操作 | この画面に来たユーザーが最初に行う操作 |
| 主要コンポーネント | ページ内の主な React コンポーネント |
| データ取得元 | Supabase テーブル / Storage / localStorage / API |
| 状態所有者 | UI 状態をどこが保持しているか |
| 重複導線 | 同じ目的地・同じ操作が別の場所にも存在する箇所 |

| 後続タスク | ロードマップの対応 ID |
| 改修リスク | 変更時の注意 |
